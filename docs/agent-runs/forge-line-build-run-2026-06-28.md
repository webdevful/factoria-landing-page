## Lessons Codified

1. **Swiper visibleMedia Audits**: When running automated QA engines that verify image assets in sliders, use actual `<img>` tags instead of CSS `background-image`. Use an absolute sibling overlay `div` to handle overlay gradients.
2. **Playwright Navigation Aborts**: Single-threaded HTTP servers (like standard python `http.server`) can cause Playwright connection aborts (`net::ERR_ABORTED`) during concurrent sub-resource requests. Running a multi-threaded static server (e.g., node-based `npx http-server` or `serve`) resolves this completely.
3. **Favicon Fallbacks**: Browsers and testing frameworks frequently request `/favicon.ico` at the root regardless of page head declarations. Putting copies of the WLFS square favicon in the public root prevents 404 console errors.
